# Electrical Factor Taxonomy
Status: Draft

## Main Factor Groups
- System basis: voltage level, source topology, load type, criticality
- Protection: breaker rating, settings, coordination, fault duties
- Asset condition: transformer gas or health, feeder condition, thermal margin
- Power quality: harmonics, voltage drop, transient sensitivity
- Safety context: arc-flash exposure, PPE boundary, maintenance mode

## Governing Questions
- What drives breaker and fault acceptance?
- What changes arc-flash severity?
- Which issues are asset health problems versus system configuration problems?
