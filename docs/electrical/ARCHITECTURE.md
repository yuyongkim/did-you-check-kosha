# Electrical Architecture
Status: Draft

## Decision Chain
- System and equipment context
- Load, fault, and power-quality evaluation
- Protection and safety screening
- Operating and maintenance package generation

## Key Factors
- Voltage class, equipment type, source and load profile
- Fault current, breaker rating, and coordination margin
- Transformer condition and harmonic environment
- Arc-flash energy and work practice boundary

## Output Package
- Load review summary
- Fault and breaker margin package
- Arc-flash package
- Operating restrictions and actions
