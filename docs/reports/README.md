This folder provides sample-filled engineering submission reports for all disciplines.

- Civil:
  - `docs/reports/SAMPLE_CIVIL_REPORT_SUBMISSION_V0.1.md`

## Usage
1. Copy the sample closest to your discipline.
2. Replace metadata, input snapshot, metric values, and evidence paths with real project data.
3. Update revision records:
   - `docs/revisions/CHANGELOG.md`
   - `docs/revisions/DELIVERY_LOG.md`
