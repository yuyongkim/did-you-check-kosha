# Instrumentation Verification Rules
Status: Draft

## Layer Checks
- Layer 1: uncertainty and interval math consistency
- Layer 2: SIS and device-selection consensus
- Layer 3: drift, SIL, and operability red-flag screening
- Layer 4: reverse checks from interval and safety outputs to basis

## Core Red Flags
- PFDavg above SIL limit
- Predicted drift exceeds tolerance
- Valve or element capacity margin exhausted
- Calibration interval unsupported by drift evidence
- Signal integrity risk unresolved
