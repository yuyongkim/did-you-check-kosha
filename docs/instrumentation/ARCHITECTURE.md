# Instrumentation Architecture
Status: Draft

## Decision Chain
- Measurement and safety basis
- Device and uncertainty evaluation
- SIS and reliability screening
- Loop and maintenance package generation

## Key Factors
- Process variable, range, and required accuracy
- Device type, installation environment, and failure mode
- SIL target, PFDavg, and proof-test interval
- Calibration history, drift trend, and maintainability

## Output Package
- Loop basis summary
- SIL and uncertainty review
- Calibration interval recommendation
- Device action and maintenance notes
