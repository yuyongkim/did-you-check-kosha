# Vessel Engineering Model
Status: Draft

## Decision Chain
- Design basis and geometry definition
- Damage mechanism and thickness evaluation
- Code and FFS screening
- Deliverable package generation for inspection and repair planning

## Main Factor Groups
- Design basis:
  - pressure, temperature, material, corrosion allowance, code case
- Geometry:
  - shell, head, nozzle, opening, support arrangement
- Damage state:
  - corrosion, local thinning, cracking, external-pressure sensitivity
- Fabrication and repair context:
  - weld joint efficiency, PWHT history, repair constraints
- Inspection context:
  - history, criticality, access, outage window

## Governing Questions
- What sets required thickness and margin?
- What triggers FFS or repair review?
- Which openings, loads, or damage states change the evaluation basis?

## Primary Outputs
- Required thickness package
- Remaining life summary
- FFS screening note
- Inspection interval recommendation
- Repair and monitoring actions

## Downstream Use
- RBI and shutdown planning
- Repair scope definition
- Inspection work-pack preparation
- Nozzle and piping interface review
