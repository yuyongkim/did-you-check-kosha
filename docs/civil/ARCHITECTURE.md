# Civil Architecture
Status: Draft

## Decision Chain
- Structural and foundation context
- Concrete durability and distress evaluation
- Capacity and serviceability screening
- Repair and monitoring package generation

## Key Factors
- Geometry, reinforcement, cover, and loading
- Exposure, carbonation, chloride or moisture environment
- Crack, spall, settlement, and damage mode
- Anchor interfaces, repairability, and outage access

## Output Package
- Foundation status summary
- Concrete distress review
- Settlement and crack risk note
- Repair and monitoring actions
