# Civil Verification Rules
Status: Draft

## Layer Checks
- Layer 1: capacity, settlement, and durability math consistency
- Layer 2: code and damage-mode consensus
- Layer 3: substantial-damage and serviceability red-flag screening
- Layer 4: reverse checks from observed distress to predicted mechanism

## Core Red Flags
- Substantial damage condition met
- Carbonation or corrosion reaches cover depth
- Crack width exceeds allowable limit
- Settlement exceeds equipment tolerance
- Spalling or anchor distress invalidates support assumption
