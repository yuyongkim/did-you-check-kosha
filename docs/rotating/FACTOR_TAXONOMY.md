# Rotating Factor Taxonomy
Status: Draft

## Main Factor Groups
- Asset basis: machine type, driver, duty, speed, criticality
- Condition state: vibration, bearing, lubrication, temperature trend
- Mechanical interface: alignment, coupling, nozzle loads, base stiffness
- Protection logic: alarm and trip thresholds, instrument confidence
- Maintenance context: history, outage window, spare strategy

## Governing Questions
- What drives health status and intervention urgency?
- Which issues are internal versus externally induced?
- What changes alarm, trip, or shutdown recommendations?
