# Rotating Verification Rules
Status: Draft

## Layer Checks
- Layer 1: metric and threshold consistency
- Layer 2: machinery acceptance consensus
- Layer 3: trip, vibration, and reliability red-flag screening
- Layer 4: trend and back-calculation consistency

## Core Red Flags
- Vibration above acceptance threshold
- Nozzle load margin exhausted
- Bearing or lubrication distress pattern
- Protection threshold inconsistent with machine service
- Foundation or alignment issue masking root cause
