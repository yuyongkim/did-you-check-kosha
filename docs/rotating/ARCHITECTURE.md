# Rotating Architecture
Status: Draft

## Decision Chain
- Asset and duty context
- Condition indicators and margin evaluation
- Protection and reliability screening
- Maintenance and monitoring package generation

## Key Factors
- Machine type, service, and criticality
- Vibration, temperature, lubrication, and bearing condition
- Alignment, nozzle load, and base or foundation condition
- Protection setpoints and trip logic

## Output Package
- Health status summary
- Protection consistency review
- Maintenance actions
- Monitoring recommendation
