# Piping Verification Rules
Status: Draft

## Layer Checks
- Layer 1: arithmetic and unit consistency
- Layer 2: standards and decision consensus
- Layer 3: red-flag and constructability screening
- Layer 4: reverse checks from deliverables back to basis

## Core Red Flags
- Spec mismatch with design basis
- Unsupported nozzle load or thermal growth
- Impossible field weld location
- Missing vent, drain, or slope requirement
- Missing QA or test boundary definition
