# Piping Factor Taxonomy
Status: Draft

## Main Factor Groups
- Process and design basis: fluid, pressure, temperature, corrosion allowance, line class
- Topology and interface: P&ID connectivity, nozzle orientation, branch logic, tie-in point
- Routing and constructability: space claim, slope, access, vent and drain requirements
- Stress and support: thermal growth, nozzle load, support spacing, vibration risk
- Fabrication and erection: spool split, shop vs field weld, transport limit, installation sequence
- Quality and maintenance: weld category, NDE scope, PWHT, test boundary, removable spool need

## Governing Questions
- What fixes the line class and schedule?
- What determines ISO geometry and spool break points?
- Which decisions must remain field-adjustable?
- Which constraints are code-driven versus construction-driven?
