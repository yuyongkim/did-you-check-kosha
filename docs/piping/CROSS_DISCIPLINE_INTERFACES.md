# Piping Cross Discipline Interfaces
Status: Draft

## Primary Interfaces
- Vessel: nozzle geometry, rating, and inspection boundary
- Rotating: nozzle load transfer, vibration coupling, alignment sensitivity
- Instrumentation: tapping points, impulse line routing, operability access
- Steel: pipe rack load, support location, deflection allowance
- Electrical: heat tracing, power routing, hazardous-area constraints
