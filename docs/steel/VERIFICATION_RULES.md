# Steel Verification Rules
Status: Draft

## Layer Checks
- Layer 1: section property and demand calculation consistency
- Layer 2: code and load-path consensus
- Layer 3: overload, deflection, and connection red-flag screening
- Layer 4: reverse checks from observed distress to predicted demand

## Core Red Flags
- Demand-capacity ratio exceeds threshold
- Deflection exceeds serviceability limit
- Connection distress or failure detected
- Corrosion loss invalidates assumed section
- Unsupported external load from piping or cable tray system
