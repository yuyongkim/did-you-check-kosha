# Steel Architecture
Status: Draft

## Decision Chain
- Structural context and loading basis
- Capacity and deflection evaluation
- Connection and degradation screening
- Repair, support, and fabrication package generation

## Key Factors
- Member geometry, material grade, and boundary condition
- Gravity, piping, tray, and equipment loads
- Corrosion loss, deflection, and connection condition
- Erection constraints and inspection accessibility

## Output Package
- Capacity status
- Serviceability summary
- Connection review
- Repair or reinforcement recommendations
