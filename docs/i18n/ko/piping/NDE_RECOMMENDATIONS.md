# 배관 재질별 NDE 추천 가이드 (스크리닝용)

## 1. 목적
이 문서는 배관 워크벤치에서 사용하는 재질 기반 NDE 추천 로직을 설명합니다.

실무 스크리닝 참고용이며, 최종 검사 범위는 사내 절차, 코드 에디션, RBI/FFS 절차를 따라야 합니다.

## 2. 재질 그룹별 추천 NDE
| 재질 그룹 | 기본 방법 | 타겟 방법 | 주요 우려 메커니즘 |
| --- | --- | --- | --- |
| 탄소강 | UT 두께 측정(CML) | PAUT/UT 맵핑, MT(선별 용접부) | 균일 감육, 국부 피팅, 지지부/데드레그 부식 |
| 저합금강 | UT + PAUT(용접/HAZ) | MT/PT, 복제금속조직(고온 운전) | 감육 + 용접부 균열 + 고온 열화 |
| 스테인리스강 | UT 맵핑 | PT, ECT(필요 시) | 용접 HAZ 인근 염화물 피팅/SCC |
| 듀플렉스강 | UT 맵핑 | PT, PMI/페라이트 확인 | 국부 부식 + 용접 품질/페라이트 관리 |
| 니켈합금 | PT(용접 루트/토우) | PAUT/TOFD, PMI 확인 | 공격성 유체에서 국부 손상 및 용접부 결함 |
| 재질 미상 | UT CML 기준선 확보 | PT + PMI 우선 | 메커니즘 특정 전 재질 불확실성 |

## 3. 권장 주기 로직
- 즉시 + 1~3개월 재점검:
  - 온도 하드 리미트 초과 또는 위험도 `critical`.
- 3~6개월 타겟 캠페인:
  - 위험도 `warning`, 온도 override 모드, 또는 부식률 상승.
- 6개월 집중 점검:
  - 중간 수준 감육 추세(심각 플래그 없음).
- 6~12개월 정기 점검:
  - 추세 안정, 위험도 `normal`.

## 4. 유체/서비스 조건 보정 힌트
- 염화물/해수:
  - 저점/데드레그 CML 밀도 증가,
  - 스테인리스/듀플렉스는 용접 HAZ, 정체 구간 PT/ECT 강화.
- Sour/H2S/Amine:
  - 균열 지향 용접 검사와 경도 확인(절차 기준) 추가.
- Steam:
  - FAC 취약 형상(엘보, 리듀서, 제어밸브 하류) 우선 점검.

## 5. 용어
- UT: 초음파 탐상
- PAUT: 위상배열 초음파 탐상
- TOFD: 시간차 회절법
- PT: 침투탐상
- MT: 자분탐상
- ECT: 와전류 탐상
- PMI: 재질 식별
- CML: 상태 모니터링 위치

## 6. 거버넌스 주의사항
- 본 가이드는 ASME/API 코드 요구사항을 대체하지 않습니다.
- 불확실한 조건은 반드시 SME 검토 및 FFS 절차로 에스컬레이션하십시오.
